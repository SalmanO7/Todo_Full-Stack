# DAPR (Distributed Application Runtime) Skill

This skill provides comprehensive guidance for building distributed microservices using DAPR (Distributed Application Runtime) from hello world examples to professional production systems.

## Overview

The DAPR skill contains:

- **SKILL.md**: Main skill file with comprehensive documentation
- **scripts/**: Utility scripts for common DAPR operations
- **references/**: Detailed reference materials
- **assets/**: Example code and templates

## Features

- Complete examples for all DAPR building blocks (service invocation, state management, pub/sub, actors, secrets)
- Configuration examples for various component types
- Kubernetes integration patterns
- Security configuration guidance
- Production best practices
- Troubleshooting assistance

## Usage

When working with DAPR-related tasks, this skill will provide:

- Code examples and templates
- Best practices for different scenarios
- Configuration guidance
- Troubleshooting assistance
- Security recommendations
- Performance optimization tips
